## Usage
<Steps>
<Step>
### Initialize
```ts
import ExpoVoicevox from "@kuzulabz/expo-voicevox";

// A system URI must be passed. ex:  Expo-Filesystem
const isSuccessful = await ExpoVoicevox.initialize(openJTalkDicDir, modelDir)
```
</Step>
<Step>
### Load Voice Model
```ts
import ExpoVoicevox from "@kuzulabz/expo-voicevox";

// You can find the styleId by grabbing the models metadata
const styleId = ExpoVoicevox.getMeta()[0]["styles"][0]["id"];

const isSuccessful = await ExpoVoicevox.loadModel(styleId)
```
</Step>
<Step>
### Generate
```ts
import ExpoVoicevox from "@kuzulabz/expo-voicevox";

const text = "無料で使える中品質なテキスト読み上げ";

// Basic TTS usage
const result = await ExpoVoicevox.tts(text, styleId)
const wavUri = `data:audio/wav;base64,${result}`;

// Advanced usage
let audioQuery = await ExpoVoicevox.createAudioQuery(text, styleId);
if (audioQuery) {
    const result = await ExpoVoicevox.synthesis({
        ...audioQuery, 
        speed_scale: audioQuery.speed_scale * 0.5
        }, speakerId);
    const wavUri = `data:audio/wav;base64,${result}`;
}
```
</Step>
</Steps>
