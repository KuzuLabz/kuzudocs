[Mecab](https://github.com/shogo82148/mecab) brought to React Native via [Nitro Modules](https://nitro.margelo.com/).

Currently supports IPADIC and UNIDIC. For other dictionaries, refer to the [initialize function](#initialize-1).

## Install

```bash tab="npm"
npm install @kuzulabz/react-native-nitro-mecab react-native-nitro-modules
```

```bash tab="pnpm"
pnpm add @kuzulabz/react-native-nitro-mecab react-native-nitro-modules
```

```bash tab="yarn"
yarn add @kuzulabz/react-native-nitro-mecab react-native-nitro-modules
```

```bash tab="bun"
bun add @kuzulabz/react-native-nitro-mecab react-native-nitro-modules
```

## Usage
<Steps>
<Step>
### Initialize
```ts
import { Mecab } from '@kuzulabz/react-native-nitro-mecab';

await Mecab.initialize('file:///path/to/dict/directory', 'ipadic');
```
</Step>
<Step>
### Tokenize
```ts
import { Mecab, Token } from '@kuzulabz/react-native-voicevox';

const text = "すもももももももものうち";

// Tokenize
const tokens: Token[] = Mecab.tokenize(text);
tokens[0].surface;

// Segment
const words: string[] = Mecab.wakati(text);

// Advanced usage
const nodes = Mecab.parseToNodes(text);
nodes[0].cost;
```
</Step>
</Steps>

## API Reference
### Setup
#### initialize
Initializes Mecab by loading the dictionary and creating a tagger.

`Unsupported Dictionaries` - Set dictType to 'unknown'. Tokens may be incorrect with this dictType. 

The parseToNode() with a custom feature parser is recommended for unsupported formats. 
For official support, please send a PR or create an issue that includes the feature order.
```ts
await Mecab.initialize("file://...", "ipadic");
```
<TypeTable 
    type={{

            type: 'string',
            required: true,
            description: 'The local uri of the dictionary directory ("file://...").'
        },
        dictFormat: {
            type: 'DictFormat',
            required: true,
            description: 'The format of dictionary.',
            typeDescriptionLink: '#dictformat'
        },
        userDictUris: {
            type: 'string[]',
            required: false,
            description: "An array of user dictionary directory uri's."
        }
    }}
/>

#### deinitialize
Unloads the dictionary and destroys the tagger. Useful for changing dictionary or loading new user dictionaries.
```ts
Mecab.deinitialize();
```

#### getIsInitialized
Check if Mecab is ready.
```ts
const isInitialized = Mecab.getIsInitialized();
```

### Core
#### tokenize
Tokenize a japanese sentence.
```ts
const tokens = Mecab.tokenize("すもももももももものうち");
tokens[0].surface
```
<TypeTable 
    type={{
        text: {
            type: 'string',
            required: true,
        },
        returns: { type: 'Token[]', required: true, typeDescriptionLink: '#token' }
    }}
/>

#### wakati
Segments/splits the japanese text by word.
```ts
const words = Mecab.wakati("すもももももももものうち");
```
<TypeTable 
    type={{
        text: {
            type: 'string',
            required: true,
        },
        returns: { type: 'string[]', required: true }
    }}
/>

#### parseToNodes
Parses the japanese text into Mecab Nodes. 
```ts
const nodes = Mecab.parseToNodes("すもももももももものうち", true);
nodes[0].cost
```
<TypeTable 
    type={{
        text: {
            type: 'string',
            required: true,

            type: 'boolean',
            required: false,

            description: 'Filters out the BOS and EOS nodes.'

#### getLattice
Parses the sentence into tokens with nbest results.
```ts
const lattice = await Mecab.getLattice("すもももももももものうち", 2);
lattice[0] // first set of tokens results
```
