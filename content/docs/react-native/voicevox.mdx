## What's VOICEVOX?
VOICEVOX is an open source text-to-speech synthesizer created by [Hiroshiba](https://github.com/Hiroshiba) and the [VOICEVOX team](https://github.com/orgs/VOICEVOX/people).

For more information, visit these links:

- [Official Site](https://voicevox.hiroshiba.jp/)
- [VOICEVOX Github](https://github.com/VOICEVOX)
- [Wiki Article](https://vocalsynth.fandom.com/wiki/VOICEVOX) (English)

## Usage
### Text-to-Speech
<Steps>
<Step>
#### Initialize
```ts
import { Voicevox } from '@kuzulabz/react-native-nitro-voicevox';

await Voicevox.initialize('file:///path/to/OpenJtalkDict/directory');
```
</Step>
<Step>
#### Load Voice Model
```ts
import { Voicevox, openVoiceModelFile } from '@kuzulabz/react-native-nitro-voicevox';

const voiceModelFile = await openVoiceModelFile('path/to/your/model.vvm');

await Voicevox.loadModel(voiceModelFile);
```
</Step>
<Step>
#### Generate
```ts
import { Voicevox } from '@kuzulabz/react-native-nitro-voicevox';

const text = "こんにちは世界！";

const metas = await Voicevox.getMetas();
const styleId = metas[0].styles[0].id;

// Basic TTS usage
// Outputs either an ArrayBuffer or a base64 string (ex: data:audio/wav;base64,...)
const base64Uri = await Voicevox.tts(text, styleId, {format: 'base64'});

// Advanced usage
let audioQuery = Voicevox.createAudioQuery(text, styleId);
audioQuery.pitch_scale += 0.1; // Modify parameters as needed

const wavArrayBuffer = await Voicevox.synthesis(audioQuery, 1, {format: 'arraybuffer', enableInterrogativeUpspeak: true});
```
</Step>
</Steps>

### Song
<Steps>
<Step>
#### Initialize
```ts
import { Voicevox } from '@kuzulabz/react-native-nitro-voicevox';

await Voicevox.initialize('file:///path/to/OpenJtalkDict/directory');
```
</Step>
<Step>
#### Load Voice Models
```ts
import { Voicevox, openVoiceModelFile } from '@kuzulabz/react-native-voicevox';

// As of 0.16.3 model release, the model called s0.vvm contains song specific voices.
const songVoiceModel = await openVoiceModelFile('path/to/song/model/s0.vvm');

// Optional. Most voices should work as a singer (sing / frame_decode type).
const voiceModel = await openVoiceModelFile('path/to/your/model.vvm');

await Voicevox.loadModel(songVoiceModel);
await Voicevox.loadModel(voiceModelFile);
```
</Step>
<Step>
#### Sing
```ts
import { Voicevox, Score } from '@kuzulabz/react-native-voicevox';

const SINGING_TEACHER = 6000;
const SINGER = 3000;

const score: Score = {
    notes: [
        { key: null, frameLength: 15, lyric: "" },
        { key: 60, frameLength: 45, lyric: "ド" },
        { key: 62, frameLength: 45, lyric: "レ" },
        { key: 64, frameLength: 45, lyric: "ミ" },
        { key: null, frameLength: 15, lyric: "" },
    ]
};

const frameAudioQuery = Voicevox.createSingFrameAudioQuery(score, SINGING_TEACHER);
const wavArrayBuffer = await Voicevox.frameSynthesis(frameAudioQuery, SINGER, 'arraybuffer');
```
</Step>
</Steps>
